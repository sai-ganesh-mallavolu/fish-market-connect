# 🐟 Fish Market Connect

Fish Market Connect is a web application developed to connect fish farmers directly with buyers. The goal of this project is to reduce the dependency on middlemen by providing a platform where farmers can showcase their products and buyers can purchase fresh fish directly from them.

The application is built using HTML, CSS, and JavaScript with a clean and responsive user interface. It allows users to browse products, search and filter fish based on different criteria, place orders, and enables farmers to manage customer orders efficiently.

---

## 🚀 Features

* Browse fresh fish products from different farmers
* Search products by fish name, farmer, or location
* Filter products based on price, location, and stock availability
* Responsive design for desktop and mobile devices
* Light and Dark mode support
* Product details including price, stock, farmer information, and description
* Place orders through an easy-to-use interface
* Farmer dashboard to manage incoming customer orders
* Local Storage support for storing products and orders
